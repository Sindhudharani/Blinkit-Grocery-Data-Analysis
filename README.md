# Blinkit Grocery Data Analysis

## Project Overview
This project analyzes Blinkit grocery delivery data to extract insights on sales trends, product performance, and customer behavior. It uses Python, SQL, Excel, and Power BI for data analysis, visualization, and dashboard creation.

## Folder Structure
- `data/` – Raw dataset used for analysis.
- `notebooks/` – Jupyter notebooks for exploratory data analysis (EDA) and visualizations.
- `scripts/` – Python scripts for preprocessing, analysis, and reusable functions.
- `dashboards/` – Power BI `.pbix` dashboard files.
- `reports/` – Project reports or presentations (optional).
- `requirements.txt` – Python dependencies.

## Key Features
1. Data cleaning and preprocessing.
2. Exploratory Data Analysis (EDA) with Python visualizations.
3. Interactive Power BI dashboards showing sales trends, top products, and performance metrics.
4. Recommendations based on data-driven insights.
